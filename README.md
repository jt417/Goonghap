## 배포 방법

### 방법 1: GitHub Pages (추천)

1. 이 저장소를 GitHub에 push
2. Settings → Pages → Source → **GitHub Actions** 선택
3. push하면 자동 빌드 & 배포
4. `https://<username>.github.io/<repo-name>/` 에서 확인

### 방법 2: Vercel

1. [vercel.com](https://vercel.com) 에서 Import Git Repository
2. Framework: **Vite** 자동 감지
3. Deploy 클릭 → 끝

### 방법 3: Netlify

1. [netlify.com](https://netlify.com) 에서 Import from Git
2. Build command: `npm run build`
3. Publish directory: `dist`
4. Deploy 클릭

## 로컬 실행

```bash
npm install
npm run dev
```
